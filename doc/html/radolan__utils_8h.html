float RDMMPerHour ( RDScanType  t,
RDDataType  value 
)

Some scan types deliver values in 1/100 mm / 5 mins and so on. This method allows calculation in values of mm/hour, as is usual. Formats, which give values in 1/100 mm / 5 mins will be calculated in mm/h, summative formats will be corrected in quantity but not time. This method does not apply to reflectivity based scans, RX, DX etc. and will return -INFINITY for those.

Parameters:
scantype
value
Returns:
the converted value or -inf if not applicable set according to the scan's header
